# LifeHub
A productivity application that helps users manage tasks, diaries, and daily motivation.  

## Tech Stack
- React.js  
- Node.js, Express.js  
- MongoDB Atlas  
- JWT for authentication  
- Tailwind CSS  

## Features
- User authentication with JWT    
- CRUD for tasks and diaries  
- Motivational quotes   

## How to Run Locally
1. Clone the repo  
2. Run `npm install`  
3. Run `npm run dev`
